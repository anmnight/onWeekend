# 吸烟间隔记录小程序

帮助用户追踪吸烟习惯，统计每日吸烟数量，了解自己的吸烟模式。

## 功能

- **手动记录** - 一键记录吸烟时间
- **扫码记录** - 扫描烟盒条码，自动匹配品牌
- **品牌管理** - 添加/删除香烟品牌
- **统计看板** - 今日统计、本周趋势、历史记录

## 技术栈

- 微信小程序原生框架
- 本地存储（Storage）

## 页面结构

```
pages/
├── index/      # 首页/记录页
├── scan/      # 扫码页
├── brands/     # 品牌管理页
└── stats/     # 统计页
```

## 设计

苹果深色系风格，参考 `design/smoking-tracker.pen`

1. 克隆项目
