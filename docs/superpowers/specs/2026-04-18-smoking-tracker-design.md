# 吸烟间隔记录小程序设计

## 概述

帮助用户追踪吸烟习惯，统计每日吸烟数量，了解自己的吸烟模式。

## 核心功能

### 1. 记录功能

**手动记录：**
- 点击"记录"按钮，记录当前时间戳
- 品牌记为"未指定品牌"

**扫码记录：**
- 扫描烟盒条码
- 匹配本地品牌库（brands.json）
  - 匹配成功：使用匹配到的品牌记录
  - 无匹配：提示用户"是否添加新品牌"
- 支持手动选择品牌

**品牌管理：**
- 查看所有已添加的品牌列表
- 添加新品牌（条码 + 品牌名）
- 删除品牌（修正错误录入）

### 2. 统计功能

**今日统计：**
- 今日已吸数量
- 今日最后一支时间
- 今日平均间隔

**历史记录：**
- 昨日及之前每日吸烟数量
- 按日期查看记录列表

**周期统计：**
- 本周每日吸烟趋势
- 本周平均每日吸烟数

## 页面结构

| 页面 | 路径 | 功能 |
|-----|------|------|
| 首页/记录页 | pages/index/index | 记录按钮、扫码入口、今日统计 |
| 扫码页 | pages/scan/scan | 扫码 + 品牌匹配 + 确认记录 |
| 品牌管理页 | pages/brands/brands | 品牌列表、添加/删除品牌 |
| 统计页 | pages/stats/stats | 今日/历史/周期统计 |

| 场景 | 参数 |
|-----|------|
| 首页→扫码页 | 无参数 |

| 品牌管理→首页 | 无参数 |

## 品牌库操作流程

### 添加品牌

```
用户扫码 → 条码无匹配 → 弹窗提示"是否添加新品牌"
  → 是：跳转品牌管理页预填条码 → 用户输入品牌名 → 保存
  → 否：记录为"未知品牌"
```

### 删除品牌

```
品牌管理页 → 滑动/点击删除 → 确认弹窗 → 删除
```

## 交互流程

### 记录流程

```
首页
  ├── 点击"手动记录" → 记录时间戳 → 显示成功 → 更新今日统计
  └── 点击"扫码" → 扫码页
                        ├── 扫码成功 → 匹配品牌库
                        │     ├── 匹配成功 → 显示品牌 → 确认记录 → 返回首页
                        │     └── 无匹配 → 提示添加品牌 → 添加/跳过
                        └── 扫码失败/取消 → 返回首页
```

## UI设计方向

### 风格
- 苹果深色系（与现有onWeekend.v1.pen保持一致）
- 背景色：#000000、#1c1c1e
- 文字色：#ffffff（主）、#8e8e93（次）
