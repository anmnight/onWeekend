# 品牌管理页面优化设计

## 概述

将品牌管理页改为卡片式布局，每张卡片以用户拍摄的香烟照片作为背景，支持扫描二维码或手动输入条码添加品牌。

## 页面结构

### 品牌卡片
- **背景**：用户拍摄的香烟照片，高斯模糊处理 + 半透明深色遮罩
- **左侧**：品牌名首字圆形图标（不同品牌不同背景色）
- **主文字**：品牌名称（大字白色）
- **副文字**：条码（灰色小字）
- **右下角**：删除按钮

### 添加品牌流程

**Step 1: 点击"+" → 弹出选项**
- 扫描二维码
- 手动输入条码

**Step 2a: 扫描二维码**
- 调用相机扫码
- 成功后进入 Step 3

**Step 2b: 手动输入条码**
- 输入框让用户输入条码
- 确认后进入 Step 3

**Step 3: 完善品牌信息**
- 输入香烟名称
- 拍摄/选择卡片背景照片
- 保存

## 数据结构

```javascript
{
  "barcode": "6901234567890",
  "brandName": "中华硬盒",
  "bgImage": "base64string或本地路径"
}
```

## 技术实现

- 模糊效果：`backdrop-filter: blur(10px)`
- 图片选择：`wx.chooseImage`
- 图片存储：base64 或本地文件路径
- 遮罩：linear-gradient 渐变覆盖
